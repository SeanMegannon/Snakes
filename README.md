# Snake Game

A simple and classic Snake game built with Python.

## Three Versions Available

### 1. Web Version (Easiest - No Installation Required!)
**File:** `snake_game.html`
- Runs directly in your web browser
- No Python or dependencies needed
- Just open the file and play!
- Beautiful animations and effects

### 2. Tkinter Version (Python - No Dependencies)
**File:** `snake_game_tkinter.py`
- Uses Python's built-in tkinter library
- Works on Python 3.14+ and all Python versions
- No extra dependencies needed!

### 3. Pygame Version (Python - Enhanced Graphics)
**File:** `snake_game.py`
- Advanced graphics and animations
- Smooth movement and particle effects
- Requires pygame installation (works on Python 3.7-3.13)

## How to Play

### Quick Start (Web Version - Easiest!):

Just open `snake_game.html` in any web browser:
- Double-click the file
- Or right-click and select "Open with" → Your favorite browser

### Tkinter Version (No Installation):

```bash
python snake_game_tkinter.py
```

### Pygame Version (Enhanced Graphics):

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the game:
```bash
python snake_game.py
```

### Controls

- **Arrow Keys**: Move the snake (Up, Down, Left, Right)
- **Space**: Restart the game after game over
- **ESC**: Quit the game

### Rules

- Use the arrow keys to control the snake (blue)
- Eat the yellow apples to grow longer and increase your score
- Avoid hitting the walls or yourself
- The game gets harder as you grow longer!

### Color Scheme

- **Snake:** Blue (always)
- **Apples/Food:** Yellow (always)

## Running from Cursor

To run the game directly from Cursor, use the terminal command:

```bash
cd C:\cursor\snake_game
python snake_game_tkinter.py
```

Or for the pygame version (if pygame is installed):

```bash
python snake_game.py
```

## Features

- Simple and intuitive controls
- Score tracking
- Game over and restart functionality
- Grid-based movement
- No dependencies required (tkinter version)

Enjoy the game!
